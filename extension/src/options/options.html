Mailpalette — Settings

General

Split inbox tabs

A tab bar at the top of Gmail. Each tab is any Gmail search (is:unread, label:Clients, from:boss@x.com…). Add, rename, and remove tabs from the gear icon in the tab bar inside Gmail.

Accounts

Learned automatically as you visit each account. Switch with g 0, g 1, g 2 (the number is the /mail/u/N index). Edit the display names if you like.

Keyboard shortcuts

Click any key cell to record a new binding. Press a single key, or start with g to build a chord (e.g. g then i). Esc cancels, Enter commits. Single-key actions drive Gmail's own toolbar, so turn on Settings → See all settings → Keyboard shortcuts on in Gmail.

Open Gmail once so the command list can load.
